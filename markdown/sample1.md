### リスト
リストは * じゃなくて - で書く
Tab（shift + Tab）でインデントできる。VS Code のショートカットも使えるので、快適にインデントが設定可能っぽい

- リスト1
    - リスト1-1
    - リスト1-2
        - リスト1-2-1
        - リスト1-2-2
- リスト2
    - リスト2-1

番号付きリストは勝手に採番してくれる。
移動やインデントもだいたいいい感じに対応してくれる。これならほぼOneNoteと同じ感覚で書けそう。

1. あは
    1. あはは
2. いひ
3. うふ
    1. うふふ
        1. うふふふ
        2. うふふふ
    2. うふふ
4. えへ
